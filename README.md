## Getting Started
### Prerequisites
Ensure you have npm installed:
```bash
npm install npm@latest -g
```
Clone the repo

```bash
git clone https://github.com/marinatea/phone_cat_front
```
Install NPM packages

```bash
npm install
```
Start
```bash
npm start
```
